# Maintain English and Chinese READMEs

The repository will use an English `README.md` as the primary index because system names, papers, venues, and public artifacts are mostly published in English and the list should be easy for international readers to search and contribute to. A Chinese translation, `README.zh-CN.md`, will be maintained alongside it and should be updated whenever the primary README changes, accepting the extra maintenance cost to make the survey useful to Chinese readers as well.

The two READMEs should stay structurally and semantically synchronized: section order, table contents, included systems, scope, exclusions, and criteria must match. The Chinese version does not need to be a sentence-by-sentence translation and may use more natural Chinese wording.
